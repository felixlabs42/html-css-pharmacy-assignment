## Project Overview

Pharmacy Website Assignment is a front-end web development project centered on a pharmacy homepage and a lab test booking page built with HTML and CSS. The design follows a retail pharmacy use case, combining product discovery, category-based navigation, and a form-driven booking path for diagnostic services.

This assignment is intended to showcase layout construction, responsive page structure, dropdown navigation, animated imagery, product card presentation, and form validation. It combines familiar site elements such as a header, footer, sticky navigation, and booking form into a single practical demonstration.
